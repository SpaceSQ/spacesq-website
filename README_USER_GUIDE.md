## 1.这是什么？
这是一个**“灵魂注入器”**。
传统的智能家居 App 让你控制开关。SpaceSQ 让你定义空间的**物理状态**。
当你运行这个脚本，你的家就不再是一堆电器，而是一个由**光、空气、声音、电磁、能源、视觉**六大要素组成的“可计算实体”。

## 2. 工作原理 (First Principles)
SpaceSQ 遵循**“黑盒原则”**：
1.  **感知：** 脚本读取你家里所有设备的状态（灯开了吗？空调几度？）。
2.  **翻译：** 脚本将这些状态翻译成物理语言（光通量增加了 800 Lux，室温降低了 2°C）。
3.  **上报：** 将物理快照上传给 SpaceSQ 云端大脑。
4.  **进化：** AI 分析这些数据，学习你的习惯，并反向微调空间环境。

## 3. 快速开始 (3步走)

### 第一步：准备环境
你需要一台 24小时开机的设备（PC、Mac、树莓派或 NAS）。
确保安装了 Python 3.8+。

### 第二步：配置你的空间 (关键!)
打开 `space_bridge.py`，找到 `MY_DEVICES` 区域。
这是最有趣的一步，你需要像上帝一样定义你的设备能力：

* **如果你有一盏灯：** 别管它叫“米家台灯”，请填入它能产生多少流明的光 (`luminousFlux`)，色温范围是多少。
* **如果你有一台电视：** 它是 `VISUAL` 设备，也是 `LIGHT` 设备（因为它发光），还是 `SOUND` 设备。

> **💡 技巧：** 不需要极其精准。不知道流明？填个大概（灯泡包装盒上通常有写，比如 800lm）。
